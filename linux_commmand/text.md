# Text Tool

## 0 Cat

### 0.0 概述

命令描述：cat命令用于查看内容较少的纯文本文件

cat [opt] [文件]

### 0.1 选项

-n --number 显示行号

-b --number-nonblank 显示行号，但是不对空白行进行编号

-s --squeeze-blank 当遇见连续两行以上的空白行，只显示一条
